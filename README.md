## 📖 Project Overview

This project is a styled authentication system built using:

* HTML
* Bootstrap 5
* Custom CSS
* JavaScript (basic validation & UI interactions)

It includes complete authentication flow UI with modern responsive design.

---

## ✨ Features Implemented

### 🔐 Authentication Pages

* Login Page
* Registration Page
* Forgot Password Page
* Reset Password Page
* Dashboard Page

### 🎨 Styling Features

* Bootstrap 5 integration
* Custom color theme
* Google Fonts (Poppins)
* Glass-style card design
* Smooth hover interactions
* Responsive layout

### ⚡ Bonus Features

* Password strength indicator
* Show/Hide password toggle
* Form validation alerts
* Button loading spinner
* Dark mode toggle
* Page transition animation
* Realistic navigation flow
